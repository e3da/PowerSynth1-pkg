--------------------------------------------------------
				Tech. Lib. Properties
--------------------------------------------------------
1. Collect device info
	- type
		o diode, transistor, cap, res
	- ports
		o drain,gate,source
		o cathode, anode
	- dimensions
		o assume rect. -- width, length, thickness
	- geometry
		o bondwire landing positions (wirebond positions) (list view)
		
	- material properties
		o thermal cond. (SiC, Si, etc.)
		
	- type dependent properties
		- Capacitor -> capacitance
		- Resistor -> inductance, resistance
		- Transistor -> verilog-a / model code
		- Diode -> verilog-a / model code
	
2. Collect bondwire info
	- bondwire shape
		o round, rect.
	- dimensions 
		o diameter if round
		o width and thickness if rect.
		o 
	- material properties
		o electrical cond.
	
3. Collect Substrate info
	- dimensions
		o trace thickness
		o isolation thickness
		o width and length of the system
	- material properties
		o trace thermal cond. and electrical resist.
		o isolation thermal cond.
		
4. Collect Baseplate information
	- Effective convection coefficient (power per unit area) (W/m^2)
	- material properties
		o thermal cond.
		
5. Collect Lead Information
	- dimensions
		o footprint (width, length)
	- material properties
		o electrical res
